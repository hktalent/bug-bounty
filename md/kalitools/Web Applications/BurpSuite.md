# Burp Suite软件包描述

Burp Suite是用于执行Web应用程序安全测试的集成平台。 其各种工具无缝集成在一起，可以支持整个测试过程，从应用程序攻击面的初始映射和分析，到查找和利用安全漏洞。Burp为您提供完全的控制，让您将高级手动技术与最先进的自动化技术结合在一起，使您的工作更快捷，更有效，更有趣。
